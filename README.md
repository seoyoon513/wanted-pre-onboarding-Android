# NewsApp
News API를 활용한 android 뉴스앱 입니다. 

*저장 기능 진행중

## 주요기능
* Retrofit을 이용한 서버 통신 결과 해당 뉴스 정보를 recycler view로 나타냈습니다
* 해당 뉴스를 클릭하면 상세 내용을 볼 수 있습다
* 해당 뉴스 클릭 시 adapter에서 글로벌 액션으로 처리하여 불필요한 코드를 줄였습니다
